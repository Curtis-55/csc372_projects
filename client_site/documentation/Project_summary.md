My website ThinkTree is an interactive code planner. In this website you can visually organize your ideas, design algorithms, and build structured workflows using draggable blocks and templates, making it easier to plan before writing actual code. Some Importaint features I have added to this website that have brought value are the dark mode selector, code/text block color picker and form submitions. The dark mode selector makes this website more accessable and allows you to choose the feel of the website. The code/text block color picker allows users to organize there blocks by color for easy organization and brainstorming. Finally the form submition allows users to give feedback to help improve the website.Throughout development, I have learned important lessons about user interface design, organizing code efficiently, and creating features that improve user experience. I also gained a better understanding of how small design choices, such as color and layout, can have a big impact on usability. Some future improvments that could be added to help site are more templates for users to choose from and maybe the addition of adding diffrent shapes for blocks for better organization.
